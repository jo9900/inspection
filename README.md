# yunuo
### 项目说明
为宁波聿诺开发的独立于c9的一套质检系统。聿诺从事二手手机回收业务，需要实现初检、复检、开单、导出、拍照等功能。安卓端和网页版同一套代码，使用vue+cordova,跨端UI框架使用的是vuetify.
## Project setup
```
npm install
```
